Before Engine Start

Preflight - Complete
Fuel Drain Tool - Stow
Passenger briefing - Complete
Seats - Adjust and Look
Seat Belt & Harness - On
Brakes - Set
Avionics Master - Off
Circuit Breakers - In
Fuel Selector - On Both
Beacon Light - On

Engine Start

Mixture - Rich
Carburetor Heat - Cold
Prime - 2/6 strike if cold
Master - On
Throttle - Open 1/4 inch
Propeller Area - Clear
Ignition - To Start
Engine RPM - 800
Oil Pressure - In Green

Before Taxi

Flaps - Up
Avionics Master - On
NAV light - (Sunset-Sunrise)On
Transponder - Select ALT
ATIS - Listen and Copy
Brakes - Release and Test
Communication - GRD or CTAF

Before Takeoff (runup)

Brakes - Set
Flight Instruments - Set
Engine Instruments - In Green
Avionics - Set properly for Flight
Mixture - Rich below 3000 degrees
Throttle - 1700 RPM
Magnetos - Check Lt and Rt
Carburetor Heat - On, check operation, then Off
Engine Instruments - In Green
Ammeter - Check
Suction Gage - Green Arc
Throttle - 800/1000 RPM
Throttle friction lock - Adjust
Flight Controls - Free & Correct
Doors & Windows - Secure
Transponder - Proper Code and select ALT
Lights - Strobes ON, Landing Light as Req., NAV Sunset to Sunrise
Pilot Heat - As Required
Brakes - Release

At takeoff

Wing Flaps - 0 degrees Normal, 10 degrees Soft field
Throttle - Set Full Power
Mixture RICH, lean above 3000 degrees
Airspeed - Rotate at 55 knots
Climb Speed - Vy 73 Knots, Vx 59 Knots

Climb

Throttle - Full Power
Mixture - Rich, Lean above 3000
Trim - Set for climb
Airspeed - Vx 59 Knots, Vy 73 Knots
Cruise Climb - 80 Knots

Cruise

Throttle - 2200 / 2450 RPM
Trim - Set
Mixture - Lean above 3000'
Landing Light - OFF

Decent

ATIS - Listen and Copy
Altimeter - Set
Throttle - Set as required
Carburator Heat - Set as required
Trim - Set as required
Mixture - Rich as required
Fuel Selector - On Both
Avionics - Set

Before Landing

Seats, seat belts - Secure
Fuel Selector - Both
Mixture - Rich
Carburator Heat - On as required
Landing Light - On as required

Go Around

Throttle - Full Power
Carburator Heat - Off
Flaps - IMMEDIATELY 20 degrees
Climb Speed - 20 Knots
Flaps up once obstacles clear

Landing

Flaps 10 degrees - Below 110 Knots
Flaps Full - Below 85 Knots
Normal App - 65 Knots
Touchdown - On mainwheels first, lower nose slightly
Braking - Minimum required

After Landing

Flaps - Up
Carburator Heat - Off
Landing Light - Off
Strobe Lights - Off

Shutdown

Avionics Master - Off
Throttle - 800 / 1000 RPM
Mixture - Idle cut-off
Magnetos - Off / key on dash
Master Switch - Off
Control Wheel Lock - Install
Record Hobbs, Tach on invoice
Tie downs - Install
